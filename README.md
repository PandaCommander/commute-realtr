# commute-realtr

Simple duproprio.com scraper to get listing by commute time and distance from metro stations.

## Setup

### Requirements
- Java 5+
- Installed [chromedriver](https://sites.google.com/a/chromium.org/chromedriver/) and added to PATH
- Maven
